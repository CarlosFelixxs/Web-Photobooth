# Web Photobooth

React/TypeScript web application prototype for a "New Gen Photobooth" experience connected to Accenture Liquid Studio themes.

The project is an older frontend case study, but it remains useful as a public example of React routing, component organization, static asset handling and Firebase hosting setup.

## Main Features

- React single-page application
- TypeScript entrypoint and typed React setup
- React Router routes for home and LinkedIn screen
- Componentized layout for logo, project benefits and screen sections
- Firebase Hosting configuration
- Google Analytics integration through `react-ga`

## Tech Stack

- React
- TypeScript
- React Router
- Styled Components
- Sass/CSS
- Firebase Hosting

## Local Setup

```powershell
npm install
npm start
```

Then open:

```text
http://localhost:3000
```

## Build

```powershell
npm run build
```
